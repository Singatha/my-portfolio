# my-portfolio

A personal portfolio to showcase side projects.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Singatha/my-portfolio.git
```

Go to the project directory

```bash
  cd my-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
